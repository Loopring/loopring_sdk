## 🚀 Quick Start
```shell
# Using npm
npm i @loopring-web/loopring-sdk --save
# Using yarn
yarn add @loopring-web/loopring-sdk
```
> Make sure you are using the original npm registry.  
> `npm config set registry http://registry.npmjs.org`
